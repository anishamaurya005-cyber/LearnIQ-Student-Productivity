# 🧠 LearnIQ - AI Student Productivity System

## 📌 Overview
LearnIQ is an AI-powered system that analyzes student behavior and predicts productivity using machine learning and rule-based intelligence.

## 🚀 Features
- Interactive UI using Streamlit
- Personalized recommendations
- Machine Learning prediction (Random Forest)
- Performance score system
- Dashboard with 6 visualizations

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas, NumPy
- Matplotlib
- Scikit-learn

## 📊 Input Features
- Study Hours
- Sleep Hours
- Screen Time
- Stress Level
- Attendance
- Assignment Score
- Exam Score

## 🎯 Output
- Productivity Score
- Performance Category
- Personalized Suggestions

pip install -r requirements.txt
